# Decision Logs

Version: 1.0

---

## Purpose

This directory stores non-architectural project decisions.

These records document product, process, UX, implementation, or organizational decisions that do not require a formal ADR.

---

## Typical Entries

- Product Decisions
- UX Decisions
- Development Process
- Naming Decisions
- Technical Notes

---

## Rules

- Decision Logs are chronological.
- Significant architectural decisions belong in ADRs.
- Temporary implementation notes should not be stored here.
