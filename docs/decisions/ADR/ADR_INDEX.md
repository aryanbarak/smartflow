# Architecture Decision Records (ADR)

Version: 1.0

---

## Purpose

This directory contains all Architecture Decision Records (ADRs) for the SmartFlow project.

Each ADR documents a significant architectural decision, its context, alternatives considered, rationale, consequences, and implementation impact.

---

## Current ADRs

- ADR-0001 – Architecture Decision Record
- ADR-0002 – Flow AI Presence Architecture
- ADR-0003 – /agent/reason Remains Local-QA-Only

Additional ADRs will be added as the project evolves.

---

## Rules

- ADR numbers are permanent.
- ADRs must never be deleted.
- If a decision changes, create a new ADR instead of modifying history.
- Every major architectural change requires an ADR.
