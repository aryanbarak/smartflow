13 — Accessibility · WCAG 2.1 AA Compliance
Color Contrast
Primary text (#E0DCFF on #07060E): 12.4:1 · AAA ✓
Secondary text on --surface-01: 5.2:1 · AA ✓
Interactive elements: min 4.5:1 · AA ✓
Focus indicators: 3:1 against adjacent ✓
Motion + Reduced
@media (prefers-reduced-motion) → all animations become opacity transitions only. Orb breathing: opacity 0.7↔1.0 (no scale). Cards: fade only. Duration halved. Experience complete.
Keyboard + Focus
Tab order: Orb → Nav items → Main content → Context cards. Focus ring: 2px solid rgba(168,151,255,0.8), 2px offset. ⌘K opens command palette from anywhere. Escape closes all overlays.
Screen Reader
aria-live on Orb state changes. Card labels are aria-label on the section. Greeting text reads aloud. Flow AI states: "Flow AI is thinking / responding / available." Navigation uses role="navigation".
Touch Targets
Min touch target: 44×44px (Apple HIG). Orb in mobile bottom bar: 48×48px tap zone regardless of visual size. Card tap areas extend to full card boundary. Nav items: 44px height minimum.
Color Blind
No information conveyed by color alone. Orb states use both color + motion. Progress uses both color + percentage text. Error uses both color + icon. Habit completion: fill + checkmark shape.
