# DailyFlow

Modern productivity and learning platform built with React, TypeScript, Supabase and Cloudflare.

## Features

- Task & productivity management
- AI learning tools
- Document management
- Music & media section
- Authentication system
- Responsive modern UI
- Cloudflare deployment
- Supabase backend integration

## Tech Stack

- React
- TypeScript
- Supabase
- Cloudflare Pages
- Tailwind CSS
- Vite
