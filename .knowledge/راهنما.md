# راهنمای استفاده از SmartFlow Knowledge Base

## این فایل چیست؟

این تنها فایل فارسی در پروژه است. همه فایل‌های دیگر به انگلیسی هستند
تا AI بهتر بتواند آن‌ها را پردازش کند. در چت با AI آزادی — می‌توانی
به فارسی، آلمانی یا انگلیسی صحبت کنی.

## نصب و راه‌اندازی اولیه

```powershell
cd C:\Projects\fiae-workspace\smartflow

# ۱. میانبرها را لود کن (یا kb-install بزن)
kb-install

# ۲. ساخت پایگاه دانش (Ollama باید در حال اجرا باشد)
kb-build
```

اگر Ollama در حال اجرا نیست ابتدا `ollama serve` بزن.

## استفاده روزانه

### صبح هر روز

```powershell
# ۱. لود context برای AI
kb-load

# ۲. فایل context_output.md را در VS Code باز می‌کند
# محتوای آن را کپی کن و در AI paste کن
```

### بعد از تغییرات مهم در پروژه

```powershell
# KB را rebuild کن
kb-build

# لاگ امروز را بساز
kb-daily
```

### جستجوی سریع

```powershell
# جستجو در پایگاه دانش
kb-query "supabase RLS pattern"
kb-query "flashcard SM-2 algorithm"
kb-query "deploy to cloudflare"
```

## استفاده با AI

### با Continue.dev (محلی — آفلاین)

```
kb-load
```

فایل `context_output.md` را باز کن — همه را کپی کن — در Continue paste کن.

### با Claude.ai یا ChatGPT

```
kb-load
```

فایل `context_output.md` را باز کن — همه را کپی کن — در چت paste کن.

```powershell

```

## نکات مهم

### زبان
- همه فایل‌های `.knowledge/docs/` به انگلیسی هستند
- همه فایل‌های `.prompts/` به انگلیسی هستند
- تو آزادی با AI به فارسی یا آلمانی چت کنی
- تنها این فایل (راهنما.md) فارسی است

### بروزرسانی Knowledge Base

هر بار که چیز مهمی در پروژه تغییر کرد:

1. فایل مربوطه در `.knowledge/docs/` را آپدیت کن
2. دوباره `kb-build` بزن
3. `CLAUDE_CONTEXT.md` و `DAILY_LOG.md` را هم آپدیت کن

### مشکلات رایج

| مشکل | راه‌حل |
|-------|--------|
| Ollama running نیست | ابتدا `ollama serve` بزن |
| ChromaDB خطا می‌دهد | `kb-build` دوباره بزن |
| Context خیلی بزرگ است | از `kb-load "سوال خاص"` استفاده کن |
